CubaseKeyCommands

Converts the Cubase XML Format of the Keycommands preference file or an exported Generic Remote file to a pretty HTML display that can be printed or saved.

The "Key Commands.xml" on Windows is in "%APPDATA%\Steinberg\Cubase 10.5_64"(or the respective version, e.g. "9.5_64"). On a Mac, look in “/Users/[username]/Library/Preferences/[program name]/”.
